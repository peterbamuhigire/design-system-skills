# Example: Font Default Refresh Note

## Suspected shift

A newly popular AI UI generator appears to be defaulting to `Example Sans` for dashboard and SaaS
mockups.

## Evidence grade

Moderate. The signal appears in vendor examples, several generated-template galleries, and
designer commentary, but there is not yet enough broad marketplace evidence for a hard ban.

## Classification

Watchlist / caution.

## Design consequence

Do not add `Example Sans` to the hard-ban list yet. Add a watchlist note if the pattern repeats;
require a deliberate human-design rationale before it is used as a primary face.

## Scope

Typography, web-app UI, dashboard artifacts.

## Date checked

2026-06-22.

## Changed files

None. Evidence is not strong enough for doctrine.
