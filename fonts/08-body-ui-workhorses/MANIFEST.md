# MANIFEST - fonts/08-body-ui-workhorses

The quiet, durable reading layer beneath a more distinctive display face: body copy, captions, labels, forms, dense UI, mobile screens, report body text, and tables. Binaries are gitignored; each family ships a license note.

**Count:** 20 families (target 12-20). **Default voice:** Quiet, durable, neutral. **Best for:** Body copy, captions, labels, forms, dense UI, mobile. **Avoid for:** Being the whole identity by itself. **Pair with:** a distinctive display face.
