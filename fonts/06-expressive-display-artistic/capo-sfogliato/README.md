# Capo Sfogliato
- Versione 1.0: versione iniziale

## Descrizione
Il carattere **Capo Sfogliato** è derivato da un [carattere tipografico di fine Ottocento](https://archive.org/details/schriftenatlasei02petz/page/n187/mode/2up) riportato nello _Schriften Atlas_ (1889) di Petzendorfer.

![image](images/cs_1.jpg)

Utilizzando le caratteristiche __ss01__ e __ss02__ è possibile avere separatamente le due parti che compongono ogni capolettera e attribuire colori diversi.

![image](images/cs_2.jpg)
 
![image](images/cs_3.jpg)

In passato una versione modificata del font era stata utilizzata per [Garamontio Capo](https://codeberg.org/m-casanova/GaramontioCapo), ma risultava poco adatta perché troppo ottocentesca.
