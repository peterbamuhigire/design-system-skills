**Count:** 17 families (target 12-20). **Default voice:** Engineered, precise, rational. **Best for:** Dashboards, admin UI, API docs, code, data tables. **Avoid for:** Expressive marketing heroes; mono for body copy. **Pair with:** a rational UI/body face with a mono accent.
