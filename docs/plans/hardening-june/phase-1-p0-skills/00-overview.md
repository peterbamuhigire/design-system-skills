## Definition of Done
- [ ] All 24 P0 skills authored in the `_TEMPLATE` format, in the right group, each citing doctrine + the relevant 2026-standards ref, **each with ≥1 worked example**.
- [ ] The P0-ten existing skills each gain their named `references/*` + `examples/*`.
- [ ] Engine count ~61 skills; per-output-type readiness re-scored — the three worst (cross-platform mobile 22, DOCX/PDF 30, tokens/handoff 34) should clear 65+.
- [ ] Re-audit checkpoint: overall ~70 on creation; ~85 once hardened.

## Why these first
They close the audit's *disqualifying* gaps — accessibility, dark mode, tokens, handoff, real
document formatting, states, i18n, performance, research, prototyping, navigation, conversion —
i.e. the things whose absence makes "world-class" not credible.
