**New skills proposed: 68.** (Existing 37 are redistributed, not re-counted.)

**Ceiling check:** 37 + 68 = **105 total**. This slightly exceeds the 75–100 target. The list is deliberately over-supplied so the user can **defer all P2 items** (≈20 of the 68) to land at **~85 skills** — comfortably inside 75–100 — while shipping every **P0** (the disqualifying gaps: accessibility, dark mode, tokens, handoff, doc formatting, states, i18n, performance, research, prototyping, conversion, navigation) and most **P1**.

**Recommended build order:** all **P0** first (≈24 skills — these close the "world-class" credibility gaps), then **P1**, then **P2** as headroom allows. The four net-new groups (09 Systems/Tokens, 10 Content, 11 Imagery, cross-cutting A11y/Ops) and the renamed group 13 (real document formatting) carry the highest marginal value per the doctrine's "looks human-made" moat.
